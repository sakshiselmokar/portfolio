📄 AI Summary
Neural networks are machine learning models loosely inspired by the human brain. They consist of layers of interconnected nodes (neurons) that transform input data through weighted connections. Overfitting occurs when a model learns training data too precisely — memorising noise rather than learning generalisable patterns — leading to poor performance on new, unseen data. Regularisation techniques like dropout and L2 penalty help prevent it.

What is AirFlow AI?
Use your index finger to draw shapes in mid-air — rectangles, circles, diamonds — via webcam
Two-finger gesture switches to text mode — write letters using EMNIST stroke recognition
Press e to trigger the intelligence pipeline
Outputs: validated graph, flow tree, generated Python code, and a step-by-step explanation
All runs locally — MediaPipe + OpenCV, no cloud required

AirFlow AI — Air Drawing → Flowchart Generator
computer vision · hand tracking · code generation
What it does
MediaPipe Hands tracks fingertip in real-time via webcam
Index finger up → draw shapes in the air (rectangles, circles, diamonds)
Two fingers → switch to text mode — write letters via EMNIST stroke model
Press e → pipeline runs: shape detection → graph build → flowchart → code gen → explanation
Outputs: validated graph, flow tree, generated Python code, step-by-step explanation
Intelligence Pipeline
HandTracker (MediaPipe)
↳ gesture_detector → detect_gesture()
↳ shape_detector → detect_shape()
↳ stroke_predict (EMNIST model)
─────────────────
run_pipeline(data)
↳ flow_builder → create_graph()
↳ flow_validator → validate_graph()
↳ build_flow_tree()
↳ code_generator → generate_code()
↳ explainer → generate_explanation()
